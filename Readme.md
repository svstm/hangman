
### Description


Hangman game in Python, that can played in the terminal.
Available on linux, windows, macos.

### Installation

Download and run.
To launch the game, all that is needed is to run the following command in the terminal:

```bash
python main.py
```

### Usage

To launch the game, all that is needed is to run the following command in the terminal:

```bash
python main.py
```
